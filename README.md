# WTF 深度强化学习
WTF 深度强化学习算法库及教程，学习并复现经典的深度强化学习算法。

深度强化学习算法使用pytorch实现，写在jupyter notebook中。用Atari Games中的Pong测试。

## 先修课程

1. 机器学习/深度学习

2. python编程

3. 经典强化学习
